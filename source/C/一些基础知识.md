# 变量

int: 整数型，在内存中占用4个字符；  
double： 双精度浮点数，占用8个字符；  
float： 浮点型，占用4个字符；  
***
函数`sizeof`给出某个类型或变量在内存中所占据的字节数。
 
tips: 我们可以使用`sizeof`函数来判断一个数组的大小，示例如下：  
```
  int main(){

      int arry[13] = {0};
      int len_arry;  
      len_arry = sizeof(arry)/sizeof(arry[0]);
      printf("size of arry = %d",len_arry); 
      return 0;
  
  }
```
*应灵活的使用该方法*
